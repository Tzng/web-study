## 常见问题？

如何在md引入图片？

图片存放在public文件夹下，引入方式如下：

```
<img src="/images/test.png" />
```

这个是引入的结果

<img src="/images/test.png" />

> 当然，很简单，就和写HTML一样，这是因为dumi支持对md文件的扩展，所以可以直接写HTML标签

试试md的语法插入图片

![test](/images/test.png)

很棒吧，这就是dumi的强大之处，不过你使用img标签的话，可以修改图片的大小，而md的语法则不行。

可以试着修改下图片的大小

<img src="/images/test.png" width="100px" />

完美！
