## 创建和嵌套组件

React 应用程序是由**组件**组成的。 一个组件是UI（用户界面）的一部分，它拥有自己的逻辑和外观。 组件可以小到一个按钮，也可以大到整个页面。

React 组件是返回标签的**JavaScript**函数：

```
function MyButton() {
  return (
    <button>I'm a button</button>
  );
}
```
